core.js: file con le funzioni per far funzionare il sistema dinamico
app.js: file con le funzioni per questa specifica app

index.php: file costantemente caricato quando devono essere visualizzate le pagine frontend

style.css: file di stile principale

settings.json: file di configurazione settings dell'app

functions: dove vengono contenute le funzioni php backend
pages: dove vengono contenute le pagine html da caricare dinamicamente
style: dove vengono contenuti tutti i file grafici
